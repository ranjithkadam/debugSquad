Theory:

This HTML code creates a form for user input using different types of input fields.

The <form> tag defines the form structure.

<label> is used to describe each input field.

<input> creates different fields like text, email, password, date, etc.

The type attribute in <input> defines what kind of data the field will accept.
<button> is used to submit the form.

Additions:

Missing name attributes on inputs ‚Äî required for submitting data.
type="age" is invalid. There‚Äôs no such input type ‚Äî use type="number" instead.
Added required where it makes sense (for basic validation).
Added method="post" in <form> for best practice (though optional).

HTML Code:

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Forms</title>
</head>

<body>
    <form action="#" method="post">

        <label for="firstName">First Name:</label>
        <input type="text" id="firstName" name="firstName" required><br><br>

        <label for="lastName">Last Name:</label>
        <input type="text" id="lastName" name="lastName" required><br><br>

        <label for="email">Email Id:</label>
        <input type="email" id="email" name="email" required><br><br>

        <label for="password">Password:</label>
        <input type="password" id="password" name="password" required><br><br>

        <label for="phoneNo">Phone No:</label>
        <input type="tel" id="phoneNo" name="phoneNo" minlength="10" maxlength="12" required><br><br>

        <label for="date">Date:</label>
        <input type="date" id="date" name="date"><br><br>

        <label for="week">Week:</label>
        <input type="week" id="week" name="week"><br><br>

        <label for="age">Age:</label>
        <input type="number" id="age" name="age" min="0"><br><br>

        <label for="dateofbirth">Date of Birth:</label>
        <input type="date" id="dateofbirth" name="dateofbirth"><br><br>

        <label for="url">URL:</label>
        <input type="url" id="url" name="url"><br><br>

        <button type="submit">Submit</button>

    </form>
</body>

</html>

 

 üìù **HTML Form Tags Notes**

‚úÖ `<form>`

* **Purpose**: It creates a form to collect user input.
* **Example**:

  ```html
  <form> ... </form>
  ```

 

 ‚úÖ `<label>`

* **Purpose**: Describes what the input is for; improves accessibility.
* **Tip**: Use `for="id"` to link it with an input.
* **Example**:

  ```html
  <label for="search">Search</label>
  ```

---

 ‚úÖ `<input>`

* **Purpose**: Creates various types of user input fields.
* **Common `type` values**:

  * `text` ‚Äì single-line text input
  * `search` ‚Äì search box
  * `radio` ‚Äì for selecting one option from a group
  * `checkbox` ‚Äì for selecting multiple options
  * `color` ‚Äì color picker
  * `range` ‚Äì slider
  * `file` ‚Äì upload file
  * `reset` ‚Äì resets the form
  * `submit` ‚Äì submits the form
* **Example**:

  ```html
  <input type="text" id="name">
  ```

---

 ‚úÖ `type="radio"`

* **Purpose**: Allows selecting **only one option** from a group.
* **Note**: Use the same `name` for all related radio buttons.
* **Example**:

  ```html
  <input type="radio" name="gender" value="male">
  ```

---

 ‚úÖ `type="checkbox"`

* **Purpose**: Allows selecting **multiple options**.
* **Example**:

  ```html
  <input type="checkbox" name="skills" value="HTML">
  ```

---
 ‚úÖ `type="color"`

* **Purpose**: Shows a color picker to choose a color.
* **Example**:

  ```html
  <input type="color" name="color">
  ```

---

 ‚úÖ `type="range"`

* **Purpose**: Creates a slider to select a value in a range.
* **Attributes**:

  * `min`, `max` ‚Äì set the limits
  * `step` ‚Äì set the increment
* **Example**:

  ```html
  <input type="range" min="1" max="10" step="1">
  ```

---

 ‚úÖ `type="file"`

* **Purpose**: Lets the user upload a file.
* **Example**:

  ```html
  <input type="file" name="resume">
  ```

---

 ‚úÖ `<button>`

* **Purpose**: A clickable button.
* **Use**: For submitting forms, or custom JavaScript actions.
* **Example**:

  ```html
  <button type="submit">Submit</button>
  ```

---

 ‚úÖ `type="reset"`

* **Purpose**: Clears all form fields and resets them to default.
* **Example**:

  ```html
  <input type="reset" value="Reset">
  ```
